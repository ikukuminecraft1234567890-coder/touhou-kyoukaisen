自作弾幕コードヘルプ

基本形
for (let i = 0; i < 10; i++) {
    // 10回実行
}

while (true) {
    // アクション中ずっと実行
}

if (条件) {
    // 条件が true の時だけ実行
}

wait(0.5) // 0.5秒待つ

一番上に複数の while (true) や for を置くと、同時に並行して動きます。古い forever() / repeat() もそのまま使えます。

エミッター側
aimAtTarget()
spawnBullet("normal", "#ff3333", 200, angle)
spawnRing("normal", "#33ffff", 180, 24)
spawnWay("normal", "#ff3333", 250, angle, 5, 20)

moveTo("center")
moveTo("right")
moveTo("left")
moveTo("farRight")
moveTo("farLeft")
moveTo("rightUp")
moveTo("leftUp")
slideTo("farRight", 0.5)

moveTo は即移動、slideTo は指定秒数で滑らかに移動します。

弾側
speed += 10
speed -= 5
speed = 300
angle += 2
angle = angle + random(-5, 5)
color = #33ffff

aimAtTarget()
homing(90)
bounce()
slow(0.5, 0)
fast(2, 0)

if (isBounced) {
    color = #ffdd33
}

弾側コードは、発射された弾1つ1つの目線で毎フレーム動きます。

変数と条件
angle      // 角度
speed      // 速度
x, y       // 現在位置
tx, ty     // 相手の位置
dist       // 相手との距離
timer      // その弾の経過秒
second     // 秒数
frame      // フレーム数
color      // 色
isBounced  // bounceで跳ねたら1

random()        // 0から1
random(100)     // 0から100
random(-5, 5)   // -5から5

条件は if (speed > 300) や if (color == #33ffff) のように書けます。

例: 移動する発射台
while (true) {
    slideTo("center", 0.5)
    wait(2)
    slideTo("farRight", 0.5)
    wait(0.7)
    slideTo("center", 0.5)
    wait(2)
    slideTo("farLeft", 0.5)
    wait(0.7)
}

while (true) {
    aimAtTarget()
    angle += random(-8, 8)
    spawnWay("normal", "#33ffff", 360, angle, 5, 12)
    wait(0.08)
}
